PL/SQL Data Types

üìå Introduction

PL/SQL supports a wide range of data types that are used to define variables, constants, and parameters in a program. These data types help in storing and manipulating different kinds of data such as numbers, characters, dates, and Boolean values. PL/SQL also supports complex and user-defined types for advanced programming needs.

üìò Common Data Types in PL/SQL

Data Type Description
NUMBER(p, s) Stores numeric values with precision (p) and scale (s)
VARCHAR2(size) Stores variable-length character strings
CHAR(size) Stores fixed-length character strings
DATE Stores date and time values
BOOLEAN Stores TRUE, FALSE, or NULL (only in PL/SQL, not in SQL)
PLS_INTEGER Efficient integer type for calculations in PL/SQL
BINARY_INTEGER Similar to PLS_INTEGER, used for integer arithmetic
LOB (BLOB, CLOB, NCLOB) Used for storing large binary or character data

üßæ Example

DECLARE
  v_name     VARCHAR2(30);
  v_salary   NUMBER(8,2);
  v_joined   DATE;
  v_status   BOOLEAN;
BEGIN
  v_name := 'Robert';
  v_salary := 55000.75;
  v_joined := SYSDATE;
  v_status := TRUE;

  DBMS_OUTPUT.PUT_LINE('Name: ' || v_name);
  DBMS_OUTPUT.PUT_LINE('Salary: ' || v_salary);
  DBMS_OUTPUT.PUT_LINE('Joined: ' || v_joined);
  DBMS_OUTPUT.PUT_LINE('Active: ' || v_status);
END;
/
