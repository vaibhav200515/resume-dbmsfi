SQL ORDER BY Clause

üìå Introduction

The ORDER BY clause in SQL is used to sort the result set of a query by one or more columns.

By default, sorting is done in ascending order (ASC). To sort in descending order (DESC), it must be specified explicitly.

This clause is useful when you want to view data in a specific order like alphabetical, numerical, or date-wise.

üìò Syntax

SELECT column1, column2, ...
FROM table_name
ORDER BY column1 [ASC|DESC], column2 [ASC|DESC], ...;

üßæ Example: Display students sorted by age in descending order

SELECT * FROM students ORDER BY age DESC;
