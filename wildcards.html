SQL Wildcards

üìå Introduction

SQL wildcards are used with the LIKE operator to search for specific patterns in column values.

They help perform flexible and powerful string matching when exact values are unknown.

Common wildcard characters:

  • % ‚Äì Matches zero or more characters
  • _ ‚Äì Matches exactly one character
  • [charlist] ‚Äì Any single character in the list (SQL Server)
  • [^charlist] ‚Äì Any single character not in the list (SQL Server)

üìò Syntax

SELECT column1, column2, ...
FROM table_name
WHERE column_name LIKE pattern;

2Ô∏è‚É£ Ends with a letter

7Ô∏è‚É£ Not in range [^a-c]

8Ô∏è‚É£ Match any single character (underscore)
