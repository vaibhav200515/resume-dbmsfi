SQL INSERT Command

üìå Introduction

The INSERT command in SQL is used to add new records (rows) to a table.

It allows you to insert values for all columns or only specific ones. It's one of the most commonly used DML (Data Manipulation Language) commands, especially after creating a table.

Use it to populate your database with initial or ongoing data like student details, customer records, products, etc.

üìò Syntax

-- Insert into all columns
INSERT INTO table_name VALUES (value1, value2, ...);

-- Insert into selected columns
INSERT INTO table_name (column1, column2, ...) 
VALUES (value1, value2, ...);

üßæ Example: Insert data into the students table

INSERT INTO students (student_id, name, age, course, admission_date)
VALUES (101, 'Rohit Sharma', 20, 'BCA', '2024-07-15');
