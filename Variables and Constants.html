PL/SQL Variables and Constants

üìå Introduction

In PL/SQL, variables and constants are used to store data temporarily during the execution of a program. Variables can change their values during execution, whereas constants hold fixed values that cannot be altered once defined.

These elements are essential in PL/SQL programming for data manipulation, control flow, and passing information between different blocks and procedures.

üìò Declaring Variables

Variables must be declared in the DECLARE section before they can be used in the BEGIN block. The syntax is:

variable_name datatype [:= initial_value];

üìò Declaring Constants

Constants are declared using the CONSTANT keyword:

constant_name CONSTANT datatype := value;

üßæ Example

DECLARE
  v_name      VARCHAR2(30);
  v_age       NUMBER(3) := 25;
  c_pi        CONSTANT NUMBER := 3.14159;
BEGIN
  v_name := 'John Doe';
  DBMS_OUTPUT.PUT_LINE('Name: ' || v_name);
  DBMS_OUTPUT.PUT_LINE('Age: ' || v_age);
  DBMS_OUTPUT.PUT_LINE('PI: ' || c_pi);
END;
/
