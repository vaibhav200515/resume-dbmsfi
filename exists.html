The EXISTS operator in SQL is used to test for the existence of any record in a subquery.

It returns TRUE if the subquery returns one or more records, and FALSE if no rows are found.

It is commonly used in combination with a WHERE clause to filter results based on related data in another table.

üìò Syntax

SELECT column_name(s)
FROM table_name
WHERE EXISTS (
  SELECT 1 FROM related_table WHERE condition
);

üßæ Example: Find students who have registered for any course

SELECT name
FROM students s
WHERE EXISTS (
  SELECT 1
  FROM registrations r
  WHERE r.student_id = s.student_id
);

This query returns the names of students who have at least one matching entry in the registrations table.
