SQL ALTER TABLE

üìå Introduction

The ALTER TABLE statement in SQL is used to modify the structure of an existing table.

You can use this command to add, delete, or modify columns, as well as to add or drop constraints like PRIMARY KEY or FOREIGN KEY.

It's a flexible command that helps in evolving database structure without needing to drop and recreate tables.

üìò Syntax

-- Add a column
ALTER TABLE table_name ADD column_name datatype;

-- Modify a column
ALTER TABLE table_name MODIFY column_name new_datatype;

-- Drop a column
ALTER TABLE table_name DROP COLUMN column_name;

üßæ Example: Add a "gender" column to the students table

ALTER TABLE students ADD gender VARCHAR(10);
