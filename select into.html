üìå Introduction

The SELECT INTO statement in SQL is used to copy data from one table into a new table.

This command creates the new table and inserts the selected data in one step. It's helpful for backing up tables or creating archive tables based on specific conditions.

üìò Syntax

-- Copy all columns into a new table
SELECT * INTO new_table
FROM existing_table;

-- Copy selected columns into a new table
SELECT column1, column2 INTO new_table
FROM existing_table
WHERE condition;

üßæ Example: Create a backup of BCA students

SELECT student_id, name, age, course
INTO bca_students_backup
FROM students
WHERE course = 'BCA';

This query creates a new table named bca_students_backup and copies only the BCA student records from the students table.
