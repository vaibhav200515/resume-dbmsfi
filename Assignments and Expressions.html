PL/SQL Assignments and Expressions

üìå Introduction

In PL/SQL, assignments and expressions are fundamental elements used to perform calculations and store results into variables. The assignment statement stores the result of an expression into a variable using the assignment operator :=.

An expression can be a combination of variables, constants, literals, and operators that return a value. These expressions can be arithmetic (e.g., a + b), logical (e.g., a > b), or even function-based (e.g., ROUND(salary)).

üìò Syntax

variable_name := expression;

This statement evaluates the expression on the right-hand side and stores the result into the variable on the left-hand side.

üßæ Example

DECLARE
  a NUMBER := 10;
  b NUMBER := 5;
  sum_result NUMBER;
  diff_result NUMBER;
  mult_result NUMBER;
  div_result NUMBER;
BEGIN
  sum_result := a + b;
  diff_result := a - b;
  mult_result := a * b;
  div_result := a / b;

  DBMS_OUTPUT.PUT_LINE('Sum: ' || sum_result);
  DBMS_OUTPUT.PUT_LINE('Difference: ' || diff_result);
  DBMS_OUTPUT.PUT_LINE('Product: ' || mult_result);
  DBMS_OUTPUT.PUT_LINE('Quotient: ' || div_result);
END;
/

Assignment Description

diff_result := a - bSubtracts b from a
mult_result := a * bMultiplies a and b
div_result := a / bDivides a by b
