SQL Aliases

üìå Introduction

An Alias in SQL is a temporary name given to a column or table, often used to make query results more readable.

Aliases are defined using the AS keyword, and they are especially useful for complex expressions or renaming column headers in the output.

They do not change the actual column or table names in the database.

üìò Syntax

-- Column Alias
SELECT column_name AS alias_name
FROM table_name;

-- Table Alias
SELECT t.column_name
FROM table_name AS t;

üßæ Example 1: Column Alias

SELECT name AS student_name, age AS student_age
FROM students;

‚úÖ Explanation

Here, the column name is shown as student_name and age is shown as student_age in the output.

üßæ Example 2: Table Alias

SELECT s.name, s.course
FROM students AS s
WHERE s.age > 18;

‚úÖ Explanation

We assigned an alias s to the table students to simplify the query and make it more readable.
