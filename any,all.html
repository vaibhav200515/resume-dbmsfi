SQL ANY & ALL

üìå Introduction

The ANY and ALL operators are used with subqueries to compare a value against a set of values returned by the subquery.

  • ANY returns TRUE if any one of the subquery values satisfies the condition.
  • ALL returns TRUE only if all subquery values satisfy the condition.

üìò Syntax

-- Using ANY
SELECT column_name
FROM table_name
WHERE column_name operator ANY (subquery);

-- Using ALL
SELECT column_name
FROM table_name
WHERE column_name operator ALL (subquery);

üßæ Example 1: Use of ANY

SELECT name
FROM students
WHERE age > ANY (
  SELECT age FROM students WHERE course = 'BCA'
);

This returns students whose age is greater than any one of the ages of BCA students.

üßæ Example 2: Use of ALL

SELECT name
FROM students
WHERE age > ALL (
  SELECT age FROM students WHERE course = 'BCA'
);

This returns students whose age is greater than all of the ages of BCA students.
