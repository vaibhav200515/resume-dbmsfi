SQL BETWEEN Operator

üìå Introduction

The BETWEEN operator in SQL is used to filter the result set within a certain range.

It selects values that fall within a specified minimum and maximum range, inclusive of both endpoints.

It can be used with numbers, text, or dates.

üìò Syntax

SELECT column1, column2, ...
FROM table_name
WHERE column_name BETWEEN value1 AND value2;

üßæ Example 1: Select students with age between 18 and 21

SELECT * FROM students
WHERE age BETWEEN 18 AND 21;

‚úÖ Explanation

This query returns all students whose age is between 18 and 21, inclusive.

üßæ Example 2: Select students admitted between two dates

SELECT * FROM students
WHERE admission_date BETWEEN '2024-06-01' AND '2024-07-30';

‚úÖ Explanation

This query retrieves all students who were admitted between June 1st and July 30th, 2024.
