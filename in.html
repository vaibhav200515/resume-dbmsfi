The IN operator in SQL is used to specify multiple possible values for a column in the WHERE clause.

Instead of using multiple OR conditions, IN provides a cleaner and more efficient way to check if a value exists in a list.

It's useful when you want to filter results to only include specific values.

üìò Syntax

SELECT column1, column2, ...
FROM table_name
WHERE column_name IN (value1, value2, ...);

üßæ Example: Select students from specific courses

SELECT * FROM students
WHERE course IN ('BCA', 'BBA');

This query returns all students who are enrolled in either the BCA or BBA course.
