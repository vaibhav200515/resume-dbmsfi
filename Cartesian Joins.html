A Cartesian Join (also known as a cross join) returns the Cartesian product of two tables.

This means that it will return all possible combinations of rows from the first table with rows from the second table.

It usually happens when there is no join condition specified in the query.

üìò Syntax

SELECT * 
FROM table1, table2;

üßæ Example: Cartesian Join between "students" and "courses"

SELECT students.name, courses.course_name
FROM students, courses;
