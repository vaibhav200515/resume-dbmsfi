SQL Dates

üìå Introduction

SQL provides powerful functions to work with date and time values stored in a database.

Date functions are useful for filtering data based on specific timeframes, formatting date values, or performing calculations like date differences.

Common SQL date types include DATE, DATETIME, TIME, and TIMESTAMP.

üìò Syntax

Filtering records by date:

SELECT * FROM table_name
WHERE date_column = 'YYYY-MM-DD';

Using built-in functions:

SELECT CURRENT_DATE;       -- returns today's date
SELECT NOW();              -- returns current date and time
SELECT YEAR(date_column);  -- extracts the year
SELECT MONTH(date_column); -- extracts the month

üßæ Example: Select students admitted after a certain date

SELECT * FROM students
WHERE admission_date > '2024-06-01';
