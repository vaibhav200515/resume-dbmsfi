SQL AVG Function

üìå Introduction

The AVG() function in SQL is used to calculate the average value of a numeric column.

It adds up all the values in the column and divides the total by the number of non-NULL entries.

This is useful for determining averages like student marks, prices, or salaries.

üìò Syntax

-- Average of a column
SELECT AVG(column_name) FROM table_name;

-- With condition
SELECT AVG(column_name) FROM table_name WHERE condition;

üßæ Example: Average Marks of BCA Students

SELECT AVG(marks) AS average_bca_marks 
FROM students 
WHERE course = 'BCA';
