SQL Aggregate Functions

üìå Introduction

Aggregate functions in SQL perform calculations on a set of values and return a single value.

These are commonly used with the GROUP BY clause to summarize or analyze data grouped by one or more columns.

Some typical use cases include calculating total sales, average marks, counting records, etc.

üìò Common Aggregate Functions

  • COUNT() ‚Äì Returns the number of rows
  • SUM() ‚Äì Returns the total sum of a numeric column
  • AVG() ‚Äì Returns the average value
  • MIN() ‚Äì Returns the smallest value
  • MAX() ‚Äì Returns the largest value

üßæ Example: Using Aggregate Functions on a Students Table

SELECT 
  COUNT(*) AS total_students,
  AVG(age) AS average_age,
  MIN(age) AS youngest,
  MAX(age) AS oldest
FROM students;
