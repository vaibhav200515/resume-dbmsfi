The GROUP BY statement in SQL groups rows that have the same values in specified columns into summary rows.

It is often used with aggregate functions like COUNT(), SUM(), AVG(), MIN(), and MAX() to perform operations on each group of data.

üìò Syntax

SELECT column_name, AGGREGATE_FUNCTION(column_name)
FROM table_name
GROUP BY column_name;

üßæ Example: Count students per course

SELECT course, COUNT(*) AS total_students
FROM students
GROUP BY course;

This query returns the number of students enrolled in each course by grouping them based on the course column.
