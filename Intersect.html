SQL INTERSECT

üìå Introduction

The INTERSECT operator in SQL is used to return the common records from two SELECT queries.

It returns only the rows that are present in both result sets and automatically removes duplicates.

Each SELECT statement must have the same number of columns and compatible data types.

üìò Syntax

SELECT column1, column2, ...
FROM table1
INTERSECT
SELECT column1, column2, ...
FROM table2;

üßæ Example: Common Students in Two Courses

SELECT student_id FROM course_math
INTERSECT
SELECT student_id FROM course_science;

Clause Description
