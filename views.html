SQL Views

üìå Introduction

A VIEW in SQL is a virtual table based on the result of a SELECT query.

It does not store data physically but acts as a stored query that simplifies complex operations.

Views help in enhancing security by restricting access to specific columns or rows of a table.

üìò Syntax

  • Create View:
    • CREATE VIEW view_name AS
SELECT column1, column2, ...
FROM table_name
WHERE condition;
  • Select from View:
    • SELECT * FROM view_name;
  • Update View: (if the view is updatable)
    • UPDATE view_name
SET column = value
WHERE condition;
  • Drop View:
    • DROP VIEW view_name;

üßæ Example: Create a View for Students in BCA Course

CREATE VIEW bca_students AS
SELECT student_id, name, age
FROM students
WHERE course = 'BCA';
