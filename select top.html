SQL SELECT TOP Command

üìå Introduction

The SELECT TOP command is used to retrieve a limited number of rows from a result set.

This is especially useful when dealing with large tables and you only want to preview a few rows.

SELECT TOP is used in SQL Server. For other databases:

  • MySQL: use LIMIT
  • Oracle: use ROWNUM or FETCH FIRST

üìò Syntax

-- SQL Server syntax
SELECT TOP number column1, column2, ...
FROM table_name
WHERE condition;

-- MySQL equivalent
SELECT column1, column2, ...
FROM table_name
LIMIT number;

üßæ Example: Get top 3 youngest students

SELECT TOP 3 name, age
FROM students
ORDER BY age ASC;
