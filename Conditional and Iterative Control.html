PL/SQL Conditional and Iterative Control

üìå Introduction

PL/SQL provides conditional and iterative control statements that help control the flow of execution. These statements allow you to make decisions and repeat tasks within a PL/SQL block.

There are two main types of control structures:

  • Conditional Control: IF-THEN, IF-THEN-ELSE, IF-THEN-ELSIF, CASE
  • Iterative Control: LOOP, WHILE LOOP, FOR LOOP

üîÄ Conditional Control

IF condition THEN
  -- statements
ELSIF another_condition THEN
  -- statements
ELSE
  -- statements
END IF;

üßæ Example

DECLARE
  score NUMBER := 75;
BEGIN
  IF score >= 90 THEN
    DBMS_OUTPUT.PUT_LINE('Grade: A');
  ELSIF score >= 75 THEN
    DBMS_OUTPUT.PUT_LINE('Grade: B');
  ELSE
    DBMS_OUTPUT.PUT_LINE('Grade: C');
  END IF;
END;
/

üîÅ Iterative Control

PL/SQL supports the following types of loops:

  • Simple LOOP
  • WHILE LOOP
  • FOR LOOP

‚úÖ FOR Loop Example

BEGIN
  FOR i IN 1..3 LOOP
    DBMS_OUTPUT.PUT_LINE('Iteration: ' || i);
  END LOOP;
END;
/

‚úÖ WHILE Loop Example

DECLARE
  i NUMBER := 1;
BEGIN
  WHILE i <= 3 LOOP
    DBMS_OUTPUT.PUT_LINE('Count: ' || i);
    i := i + 1;
  END LOOP;
END;
/

‚úÖ SIMPLE LOOP Example

DECLARE
  i NUMBER := 1;
BEGIN
  LOOP
    DBMS_OUTPUT.PUT_LINE('Step: ' || i);
    i := i + 1;
    EXIT WHEN i > 2;
  END LOOP;
END;
/
