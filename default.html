SQL DEFAULT Constraint

üìå Introduction

The DEFAULT constraint in SQL is used to assign a default value to a column when no value is specified during an insert.

This helps in ensuring that all rows have valid data, even when the user omits a value for that column.

It can be applied to any data type and makes data entry easier and more consistent.

üìò Syntax

CREATE TABLE table_name (
  column1 datatype DEFAULT default_value,
  column2 datatype,
  ...
);

To add a DEFAULT constraint to an existing table:

ALTER TABLE table_name
ALTER COLUMN column_name SET DEFAULT default_value;

üßæ Example: Set default admission date

CREATE TABLE students (
  student_id INT PRIMARY KEY,
  name VARCHAR(100),
  admission_date DATE DEFAULT CURRENT_DATE
);
