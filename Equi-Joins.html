Querying Multiple Tables using Equi-Joins

üìå Introduction

An Equi-Join is a type of join that combines rows from two or more tables based on a related column between them using the equality operator (=).

It is one of the most commonly used joins when querying multiple tables, and it helps retrieve related data that is stored across different tables.

Equi-Joins are typically used with primary and foreign key relationships.

üìò Syntax

SELECT table1.column1, table2.column2, ...
FROM table1, table2
WHERE table1.common_column = table2.common_column;

üßæ Example: Join "students" and "departments" tables

SELECT students.name, departments.dept_name
FROM students, departments
WHERE students.dept_id = departments.dept_id;
