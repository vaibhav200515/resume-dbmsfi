SQL RIGHT JOIN

üìå Introduction

RIGHT JOIN returns all records from the right table, and the matched records from the left table.

If there is no match, the result is NULL on the side of the left table.

This is helpful when you want to see all entries from the second (right) table regardless of whether they have related data in the first (left) table.

üìò Syntax

SELECT columns
FROM table1
RIGHT JOIN table2
ON table1.column_name = table2.column_name;

üßæ Example: RIGHT JOIN students and courses

SELECT students.name, courses.course_name
FROM students
RIGHT JOIN courses
ON students.course_id = courses.course_id;

‚úÖ Explanation

This query retrieves all courses, along with the names of students enrolled in them. If a course has no students enrolled, the student name will appear as NULL.
