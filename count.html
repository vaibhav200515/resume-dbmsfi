SQL COUNT Function

üìå Introduction

The COUNT() function in SQL is used to count the number of rows in a result set.

It can be used with * to count all rows, or with a specific column to count only non-NULL values in that column.

This is especially useful in reporting, filtering grouped data, and checking the size of a dataset.

üìò Syntax

-- Count all rows
SELECT COUNT(*) FROM table_name;

-- Count non-null values in a column
SELECT COUNT(column_name) FROM table_name;

-- With condition
SELECT COUNT(*) FROM table_name WHERE condition;

üßæ Example: Count Students Enrolled in BCA

SELECT COUNT(*) AS total_bca_students 
FROM students 
WHERE course = 'BCA';
