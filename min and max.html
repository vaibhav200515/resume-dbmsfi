SQL MIN and MAX Functions

üìå Introduction

The MIN and MAX functions in SQL are used to find the smallest and largest values in a column, respectively.

These functions are useful for analyzing data, such as finding the youngest/oldest student, cheapest product, or latest date.

üìò Syntax

-- Find the smallest value
SELECT MIN(column_name) FROM table_name;

-- Find the largest value
SELECT MAX(column_name) FROM table_name;

-- With condition
SELECT MIN(column_name)
FROM table_name
WHERE condition;

üßæ Example: Find Min and Max Age of Students

  MIN(age) AS youngest,
