SQL AUTO_INCREMENT

üìå Introduction

The AUTO_INCREMENT attribute in SQL is used to generate a unique number automatically when a new record is inserted into a table.

This is commonly used for primary key columns so that each row gets a unique identifier without manually entering a value.

It is supported in databases like MySQL and MariaDB. In SQL Server, the equivalent is IDENTITY, and in PostgreSQL, you can use SERIAL.

üìò Syntax

CREATE TABLE table_name (
  column_name INT AUTO_INCREMENT,
  ...
  PRIMARY KEY (column_name)
);

To insert values while using AUTO_INCREMENT:

INSERT INTO table_name (column2, column3, ...)
VALUES (value2, value3, ...);

üßæ Example: AUTO_INCREMENT on student_id

CREATE TABLE students (
  student_id INT AUTO_INCREMENT,
  name VARCHAR(100),
  course VARCHAR(50),
  PRIMARY KEY (student_id)
);

Column Attribute Description
