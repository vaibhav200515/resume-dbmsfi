SQL UNION

üìå Introduction

The UNION operator is used to combine the result-set of two or more SELECT statements.

Each SELECT must have the same number of columns, in the same order, and with compatible data types.

By default, UNION removes duplicate rows. If you want to include duplicates, use UNION ALL.

üìò Syntax

SELECT column1, column2, ...
FROM table1
UNION
SELECT column1, column2, ...
FROM table2;
-- To include duplicates
SELECT column1, column2, ...
FROM table1
UNION ALL
SELECT column1, column2, ...
FROM table2;

üßæ Example: UNION two student tables

SELECT name FROM students_2024
UNION
SELECT name FROM students_2025;

This query combines names of students from both years, removing duplicates.
