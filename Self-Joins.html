SQL SELF JOIN

üìå Introduction

The SELF JOIN in SQL is a join where a table is joined with itself. It is used to compare rows within the same table based on a related column.

This is useful for hierarchical data or to find relationships among rows in the same table. You must use aliases to distinguish between the two instances of the table.

üìò Syntax

SELECT a.column_name, b.column_name
FROM table_name a
JOIN table_name b
ON a.common_column = b.common_column;

üßæ Example: Employees and Their Managers

SELECT e1.name AS Employee, e2.name AS Manager
FROM employees e1
JOIN employees e2
ON e1.manager_id = e2.id;

Clause Description

e1.name AS EmployeeReturns the name of the employee
e2.name AS ManagerReturns the name of the manager
