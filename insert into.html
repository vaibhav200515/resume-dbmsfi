SQL INSERT INTO

üìå Introduction

The INSERT INTO command in SQL is used to add new records (rows) to a table.

It allows inserting values into all columns or just selected ones. This is a key part of the Data Manipulation Language (DML) in SQL, used for populating tables with data such as users, products, or student details.

The command is typically used after creating a table, or when adding new data during the application workflow.

üìò Syntax

-- Insert into all columns (order must match table structure)
INSERT INTO table_name 
VALUES (value1, value2, ...);

-- Insert into selected columns
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);

üßæ Example: Insert a new student record

INSERT INTO students (student_id, name, age, course, admission_date)
VALUES (104, 'Neha Singh', 22, 'BBA', '2024-08-01');
