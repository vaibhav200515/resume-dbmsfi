SQL HAVING

üìå Introduction

The HAVING clause in SQL is used to filter groups created by the GROUP BY clause.

While WHERE filters rows before grouping, HAVING filters after aggregation. It's commonly used with aggregate functions like SUM(), AVG(), COUNT(), etc.

üìò Syntax

SELECT column_name, AGGREGATE_FUNCTION(column_name)
FROM table_name
GROUP BY column_name
HAVING condition;

üßæ Example: Show courses with more than 2 students

SELECT course, COUNT(*) AS total_students
FROM students
GROUP BY course
HAVING COUNT(*) > 2;

‚úÖ Explanation

This query groups the students by their course and only includes those groups that have more than 2 students.
