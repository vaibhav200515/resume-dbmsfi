SQL AND Operator

üìå Introduction

The AND operator in SQL is used to combine multiple conditions in a WHERE clause.

A row is returned only if all the specified conditions are true.

This operator is useful when filtering data based on multiple criteria, such as checking both age and course, or status and date, etc.

üìò Syntax

SELECT column1, column2, ...
FROM table_name
WHERE condition1 AND condition2 AND ...;

üßæ Example: Select students older than 18 and enrolled in BCA

SELECT * FROM students 
WHERE age > 18 AND course = 'BCA';

Condition Description
