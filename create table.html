üìå Introduction

The CREATE TABLE statement in SQL is used to create a new table in a database.

Each table consists of columns with specified names and data types. You can also define constraints like PRIMARY KEY, NOT NULL, UNIQUE, etc.

Tables are fundamental components of any relational database and hold the actual data in rows and columns.

üìò Syntax

CREATE TABLE table_name (
  column1 datatype constraint,
  column2 datatype constraint,
  ...
);

üßæ Example: Create a "students" table

CREATE TABLE students (
  student_id INT PRIMARY KEY,
  name VARCHAR(100) NOT NULL,
  age INT,
  course VARCHAR(50),
  admission_date DATE
);
