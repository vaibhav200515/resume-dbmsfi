SQL UPDATE Command

üìå Introduction

The UPDATE command in SQL is used to modify existing records in a table.

You can update one or multiple columns for a specific row or for multiple rows that meet a given condition using the WHERE clause.

If you omit the WHERE clause, all rows in the table will be updated, so use it carefully.

üìò Syntax

-- Update one column
UPDATE table_name
SET column1 = value1
WHERE condition;

-- Update multiple columns
UPDATE table_name
SET column1 = value1,
    column2 = value2
WHERE condition;

üßæ Example: Update a student's course

UPDATE students
SET course = 'MBA'
WHERE student_id = 103;
