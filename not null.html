SQL NOT NULL Constraint

üìå Introduction

The NOT NULL constraint in SQL ensures that a column cannot have a NULL value.

This means that every time you insert a record into the table, you must provide a value for that column.

It is commonly used to make sure important fields like names, IDs, or dates are always filled out.

üìò Syntax

CREATE TABLE table_name (
  column1 datatype NOT NULL,
  column2 datatype,
  ...
);

You can also add NOT NULL to an existing column:

ALTER TABLE table_name
MODIFY column_name datatype NOT NULL;

üßæ Example: Apply NOT NULL on "name" column

CREATE TABLE students (
  student_id INT,
  name VARCHAR(100) NOT NULL,
  age INT
);
