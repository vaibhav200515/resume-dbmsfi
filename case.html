SQL CASE Statement

üìå Introduction

The CASE statement in SQL is used to implement conditional logic within queries, similar to an IF-ELSE structure in programming languages.

It allows you to return specific values based on conditions, making query results more readable and meaningful. You can use it inside SELECT, UPDATE, ORDER BY, and other SQL clauses.

It's commonly used for creating labels, applying logic to calculations, and performing dynamic grouping or categorization in reports.

üìò Syntax

CASE
  WHEN condition1 THEN result1
  WHEN condition2 THEN result2
  ...
  ELSE default_result
END

üî∏ You can use the CASE expression inside a SELECT statement to create conditional outputs for a column.

üßæ Example 1: Grade Evaluation

SELECT name, marks,
  CASE 
    WHEN marks >= 90 THEN 'A'
    WHEN marks >= 75 THEN 'B'
    WHEN marks >= 60 THEN 'C'
    ELSE 'F'
  END AS grade
FROM students;

‚úÖ Explanation

This query assigns a grade (A, B, C, F) based on student marks using the CASE statement.

üßæ Example 2: Salary Level

SELECT emp_id, name, salary,
  CASE 
    WHEN salary > 100000 THEN 'High'
    WHEN salary BETWEEN 50000 AND 100000 THEN 'Medium'
    ELSE 'Low'
  END AS salary_level
FROM employees;

‚úÖ Explanation

The CASE statement categorizes each employee's salary into 'High', 'Medium', or 'Low' levels.
