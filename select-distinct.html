The SELECT DISTINCT command in SQL is used to return only unique (different) values from a column or combination of columns.

It helps remove duplicate data from query results, especially when dealing with large datasets where repetition might occur.

This is useful for identifying unique entries like course names, cities, or product categories.

üìò Syntax

FROM table_name;

üßæ Example: Retrieve distinct courses from students table
