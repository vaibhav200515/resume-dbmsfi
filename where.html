SQL WHERE Clause

üìå Introduction

The WHERE clause in SQL is used to filter records that meet a specific condition.

It is used with SELECT, UPDATE, DELETE, and other SQL statements to retrieve or modify only the rows that satisfy the given criteria.

Without WHERE, commands affect or return all rows in the table, which can be risky or inefficient.

üìò Syntax

SELECT column1, column2, ...
FROM table_name
WHERE condition;

üßæ Example: Select students older than 18

SELECT * FROM students WHERE age > 18;
