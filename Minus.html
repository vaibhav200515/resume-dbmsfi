SQL MINUS

üìå Introduction

The MINUS operator in SQL returns the rows from the first SELECT query that are not present in the second SELECT query.

This is useful when you want to find data that exists in one dataset but not in another.

The number of columns and their data types must match in both SELECT queries.

Note: Some database systems like MySQL do not support MINUS, but alternatives like LEFT JOIN with WHERE IS NULL or NOT IN can be used instead.

üìò Syntax

SELECT column1, column2, ...
FROM table1
MINUS
SELECT column1, column2, ...
FROM table2;

üßæ Example: Students in Math but not in Science

SELECT student_id FROM course_math
MINUS
SELECT student_id FROM course_science;
